# Iconos para PWA - Cuentas Casa

Para completar la configuración PWA, necesitas generar los siguientes iconos:

## Iconos requeridos:

1. **icon-192.png** (192x192px)
   - Icono principal para Android
   - Formato: PNG con fondo transparente o sólido
   - Debe ser cuadrado y simple

2. **icon-512.png** (512x512px)  
   - Icono de alta resolución para Android
   - Se usa para splash screen
   - Mismo diseño que el de 192px pero en mayor resolución

3. **icon-apple-touch.png** (180x180px)
   - Icono específico para iOS/Safari
   - Debe tener esquinas redondeadas (iOS las aplica automáticamente)

4. **favicon.ico** (16x16, 32x32, 48x48px)
   - Icono para navegadores
   - Múltiples tamaños en un solo archivo

## Recomendaciones de diseño:

- **Tema**: Calculadora, casa, o símbolo de servicios públicos
- **Colores**: Azul (#1976d2) como color principal (tema de la app)
- **Estilo**: Simple, icónico, reconocible en tamaños pequeños
- **Fondo**: Preferiblemente con color sólido para mejor visibilidad

## Herramientas sugeridas:

- **Online**: [PWA Icon Generator](https://www.pwabuilder.com/imageGenerator)
- **Gratuitas**: GIMP, Paint.NET
- **Profesionales**: Adobe Illustrator, Figma

## Ejemplo de prompt para IA:

"Crea un icono para una app de calculadora de servicios públicos domésticos. Estilo minimalista, color azul #1976d2, fondo blanco, que incluya elementos de casa y calculadora."

Una vez tengas los iconos, colócalos en esta carpeta `/icons/` y la PWA estará lista para instalar.
