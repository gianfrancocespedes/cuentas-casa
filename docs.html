Preguntas Frecuentes

Respuestas a las dudas más comunes

¿Necesito internet para usar la app?

No. Una vez instalada como PWA, funciona completamente offline. Solo necesitas internet para la primera instalación.

¿Se guardan mis datos?

Sí, todos los datos se guardan localmente en tu dispositivo. Puedes exportar backups para mayor seguridad.

¿Cómo calcula la distribución del agua?

El costo total del agua se divide proporcionalmente según el número de ocupantes en cada departamento.

¿Puedo modificar los cálculos?

Los cálculos son automáticos basados en los datos que ingresas. Puedes cargar cálculos anteriores del historial para modificarlos.
